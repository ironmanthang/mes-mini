Quality Control
Flow 5: Gate between Production WH and Sales WH. Pass → IN_STOCK. Fail → SCRAPPED.
APIs needed
  • GET /api/qc/pending → batches awaiting inspection
  • GET /api/qc/batches/:batchId/instances → list of product instances
  • POST /api/qc/inspect { serialNumber, result:"PASS"|"FAIL", notes?, checklist? }
  • PASS → ProductInstance moves to Sales WH (status: IN_STOCK)
  • FAIL → ProductInstance moves to Error WH (status: SCRAPPED)
